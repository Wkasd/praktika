# Step-by-Step Guide

# 1. Установка зависимостей фронтенда

# 1.1 npm install
# 1.2 Установка зависимостей бэкенда

# cd telran-backend-main
# npm install
# cd ..
# Запуск бэкенд-сервера

# cd telran-backend-main
# npm run dev
# Это запускает сервер на порту 3333
# Оставьте этот терминал открытым
# Запуск фронтенд-приложения (в новом терминале)

# npm start
# Это запускает приложение React на порту 3000 (или следующем доступном порту)
# Если порт 3000 занят, приложение запросит использование другого порта — введите 'y'
# Доступ к приложению

# Откройте браузер и перейдите по адресу http://localhost:3000 (или указанному порту)
# API бэкенда доступен по адресу http://localhost:3333

# Устранение неполадок:
# Если порты заняты, фронтенд запросит другой порт
# Убедитесь, что оба сервера запущены в отдельных терминалах
# Проверьте вывод терминала на наличие сообщений об ошибках.
