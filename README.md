# Step-by-Step Guide

# 1. Установка зависимостей фронтенда

# 1.1 npm install

# 1.2 Установка зависимостей бэкенда
# 1.2.1 cd telran-backend-main
# 1.2.2 npm install
# 1.2.3 cd ..

# 1.3 Запуск бэкенд-сервера
# 1.3.1 cd telran-backend-main
# 1.3.2 npm run dev (Это запускает сервер на порту 3333)
# 1.3.3 Оставьте терминал открытым

# 1.4 Запуск фронтенд-приложения (в новом терминале)
# 1.4.1 npm start (Это запускает приложение React на порту 3000 ((или следующем доступном порту))
# 1.4.2 Если порт 3000 занят, приложение запросит использование другого порта — введите 'y'
# Доступ к приложению получен!

# Откройте браузер и перейдите по адресу http://localhost:3000 (или указанному порту в терминале)
# API бэкенда доступен по адресу http://localhost:3333

# Устранение неполадок:
# Если порты заняты, фронтенд запросит другой порт в терминале
# Убедитесь, что оба сервера запущены в отдельных терминалах
# Проверьте вывод терминала на наличие сообщений об ошибках.
